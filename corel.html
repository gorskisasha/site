Самостоятельная работа

1. Создайте новый документ.

2. Параметры документа:

• формат страницы А4, ориентация Portrait (Книжная);

• единицы измерения миллиметры;

• шаг курсорных клавиш 0,1 мм.

3. Используя инструменты для построения стандартных объектов и команды меню Arrange (Упорядочить), создайте предложенные изображения. Желательно преобразовать каждую фигуру в один объект.

4. Сохраните документ для демонстрации преподавателю.
